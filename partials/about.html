About Crop Scheduler
Crop Scheduler has been started as B-tech project in IIIT Hyderabad. It was proposed by respected faculty of IIIT Hyd P.Krishna Reddy pkreddy@mail.iiit.ac.in. This Web Application will propose a yearly crop plan which will provide maximum profit while attaining minimum risk. Shoaib Khan, Kshitij Prasad and Rohan Gupta with the help of mentor P.Krishna Reddy have solved this problem and implemented it.
